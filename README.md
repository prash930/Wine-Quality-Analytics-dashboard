# 🍷 Wine Quality Analytics Dashboard

An interactive data analytics dashboard built with **Streamlit** to explore wine quality datasets.

## 🚀 Features

* 📊 Interactive visualizations (Plotly)
* 📈 Distribution & correlation analysis
* 🔬 Feature exploration (scatter, histograms, radar)
* 🎛️ Dynamic filters (wine type, quality, alcohol)
* 📥 Download filtered dataset

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly

## 📂 Dataset

Wine quality dataset containing physicochemical properties and quality scores for red and white wines.

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app1.py
```
